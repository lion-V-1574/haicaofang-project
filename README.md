# 海草房信息可视化系统

## 项目介绍

这是一个基于Web的海草房信息可视化系统，展示了海草房的历史沿革、结构工艺、分布地图、数据分析和文化传承等内容。

## 系统功能

- 📊 **历史沿革** - 展示海草房的发展历史和演变过程
- 🏗️ **结构工艺** - 详细介绍海草房的建筑结构和工艺特点
- 🗺️ **分布地图** - 基于百度地图的海草房真实地理分布展示
- 📈 **数据分析** - 多维度数据分析和图表展示
- 🎭 **文化传承** - 海草房文化价值和保护现状
- 🌟 **体验探索** - 互动式体验和探索功能

## 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **图表库**: ECharts 5.4.3
- **地图服务**: 百度地图API
- **UI框架**: Bootstrap 5
- **服务器**: Python HTTP Server

## 快速启动

### 方法一：使用启动脚本（推荐）

1. 确保已安装Python 3.x
2. 运行启动脚本：
   ```bash
   python start_server.py
   ```
3. 浏览器将自动打开系统页面，访问地址：http://127.0.0.1:8080/index2.html

### 方法二：手动启动

1. 在项目根目录下启动HTTP服务器：
   ```bash
   python -m http.server 8080
   ```
2. 在浏览器中访问：http://127.0.0.1:8080/index2.html

## 使用说明

1. **导航功能**: 点击顶部导航栏可快速跳转到对应模块
2. **内容隔离**: 每个模块独立显示，只在点击对应导航项时显示内容
3. **地图交互**: 在分布地图页面可查看海草房村落位置和详情
4. **数据筛选**: 数据分析页面提供多种筛选功能
5. **响应式设计**: 支持不同屏幕尺寸的显示

## 注意事项

- 系统需要互联网连接以加载百度地图API和ECharts库
- 建议使用现代浏览器（Chrome、Firefox、Edge等）
- 首次加载可能需要几秒钟时间加载地图和图表资源

## 故障排除

### 常见问题

1. **地图加载失败**
   - 检查网络连接
   - 点击"重新加载"按钮
   - 检查百度地图API密钥是否有效

2. **页面无法访问**
   - 确认服务器已启动
   - 检查端口8080是否被占用
   - 尝试使用其他端口（修改start_server.py中的PORT变量）

3. **图表显示异常**
   - 刷新页面重试
   - 检查浏览器控制台是否有错误信息

# 🌊 海草房 · 向海而生

> 山东荣成海草房濒危指数可视化分析平台 — 以数字化手段守护千年渔家智慧

---

## 📖 项目介绍

海草房是山东荣成沿海地区独有的传统民居，以大叶海草为顶、花岗岩石块为墙，冬暖夏凉、防潮耐腐，是渔民千百年来与自然和谐共生的智慧结晶。这种建筑形式距今已有 **7000年** 历史，现存最古老的单体建筑超过 **300年**，是中国沿海地区极为罕见的活态建筑遗产。

然而随着现代化进程加快，海草房正面临材料断供、工匠老龄化、村落空心化等多重威胁，保护形势严峻。

本项目基于 **ESA 濒危评估模型**，对荣成地区现存约 **650间** 海草房进行系统量化分析，覆盖烟墩角村、东墩村、小西村、东褚岛等 **4个** 核心村落，通过多维度可视化手段呈现这一濒危建筑遗产的现状、成因与保护路径，希望以数字化方式唤起更多人对海草房的关注与保护意识。

## 🛠️ 技术栈

- **前端**：HTML5、CSS3、JavaScript
- **图表**：ECharts — 数据可视化
- **地图**：高德地图 API — 村落地理分布
- **3D 渲染**：Three.js + GLTF 模型 — 建筑三维展示
- **设计系统**：渔村手记 × 潮间带（自定义 CSS 设计系统）
- **字体图标**：Font Awesome、Google Fonts（Noto Serif SC / Cormorant Garamond）

## 🚀 启动方式

本项目推荐使用 **VS Code Live Server 插件** 打开，无需任何环境配置。

1. 用 VS Code 打开项目文件夹
2. 在左侧资源管理器中找到 `index2.html`
3. 右键点击 → 选择 **Open with Live Server**
4. 浏览器自动打开，即可使用全部功能

> 💡 未安装 Live Server？在 VS Code 扩展商店搜索 `Live Server`（作者 Ritwick Dey）安装后即可使用。

> ⚠️ 请勿直接双击 HTML 文件用浏览器打开，部分资源路径在本地文件协议下会加载失败，必须通过 Live Server 启动。

## ⚠️ 注意事项

## 📌 常见问题

**地图不显示？**
检查网络连接是否正常，高德地图 API 需要联网才能加载，离线环境下地图区域会显示为空白。

**3D 模型加载失败？**
确认 `images/` 目录下的 `.glb` 和 `.gltf` 及对应 `.bin` 文件完整存在，刷新页面重试。

**页面样式错乱或功能异常？**
必须通过 Live Server 启动，不能直接用浏览器打开本地 HTML 文件。

**图表显示不完整？**
尝试调整浏览器窗口大小触发重绘，或按 F5 刷新页面。

## 🌿 设计理念

项目整体采用「渔村手记 × 潮间带」设计风格，以麻纸底色、墨色文字、海草绿与沙滩金为主色调，融合手绘线稿、墨迹晕染动画与竖排书法排版，在数据可视化的同时还原海草房的人文气息与历史质感。

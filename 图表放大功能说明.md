## 🎨 使用方法

### 对于访客（非常简单）

#### 方法 1：双击图表（推荐）
```
1. 找到任意图表
2. 双击图表区域
3. 图表立即放大显示
```

#### 方法 2：单击图表
```
1. 鼠标悬停在图表上
2. 看到右上角的 🔍 图标
3. 单击图表的画布区域
4. 图表放大显示
```

### 查看放大的图表

**所有交互功能都保留：**
- 鼠标悬停 → 显示详细数据
- 点击图例 → 显示/隐藏数据系列
- 拖拽缩放 → 查看局部数据（如果图表支持）
- 所有原有功能 → 完全可用

### 关闭放大视图

**4 种方式：**
1. 点击右上角 × 按钮
2. 点击底部"关闭"按钮
3. 点击图表外的黑色区域
4. 按键盘 ESC 键

## 🔧 技术细节

### 工作原理

1. **自动识别**
   - 页面加载 2 秒后自动扫描
   - 识别所有 ECharts 实例

3. **图表复制**

   - 创建新的 ECharts 实例
   - 保持所有交互功能

4. **自适应大小**
   - 监听窗口大小变化
   - 自动调整图表尺寸
   - 保持最佳显示效果

5. **关闭清理**
   - 销毁放大的图表实例
   - 释放内存资源
   - 恢复页面滚动

## 🎨 自定义配置

### 修改放大后的图表大小

找到 CSS 中的这一行：

```css
.chart-lightbox-content {
    width: 95%;  /* 宽度 */
    height: 90%; /* 高度 */
}
```

改成：
```css
width: 100%;  /* 更宽 */
height: 95%;  /* 更高 */
```

---

### 修改背景颜色

找到 CSS 中的这一行：

```css
.chart-lightbox {
    background-color: rgba(0, 0, 0, 0.95); /* 黑色，95% 不透明 */
}
```

改成：
```css
background-color: rgba(0, 0, 0, 0.8); /* 更透明 */
```

---

### 修改标题字体大小

找到 JavaScript 中的这几行：

```javascript
if (enlargedOption.title && enlargedOption.title[0]) {
    enlargedOption.title[0].textStyle.fontSize = 24; // 标题字体
}
```

改成你想要的大小：
```javascript
enlargedOption.title[0].textStyle.fontSize = 28; // 更大
```

---

### 禁用某个图表的放大功能

给图表容器添加类名：

```html
<div id="myChart" class="no-expand" style="height: 400px;"></div>
```

然后修改 JavaScript 代码：

```javascript
// 在 initChartLightbox 函数中
chartContainers.forEach(container => {
    // 添加这一行
    if (container.classList.contains('no-expand')) return;
    
    // ... 其他代码
});
```

**放大后：**

---

### 2. 用户引导

**提示用户可以放大：**

方法 1：在图表标题旁添加提示
```html
<h3 class="chart-title">
    濒危指数雷达图
    <span style="font-size: 12px; color: #999;">（双击放大查看）</span>
</h3>
```

方法 2：添加工具提示
```javascript
// 给图表容器添加 title 属性
container.setAttribute('title', '双击放大查看详细数据');
```

---

### 3. 性能优化

**如果图表很多：**

1. 延迟初始化
```javascript
// 只在用户滚动到图表区域时初始化
const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            initChartLightbox();
            observer.disconnect();
        }
    });
});
```

2. 按需加载
```javascript
// 只在用户第一次点击时初始化
container.addEventListener('click', function init() {
    initChartLightbox();
    container.removeEventListener('click', init);
}, { once: true });
```

## 🚨 常见错误及解决方法

### 错误 1：404 Not Found

**错误信息：**
```
GET http://localhost:8084/images/草房子结构图1.gltf 404 (Not Found)
```

**原因：** 文件路径不对或服务器没启动

**解决方法：**
1. 确认使用 `python start.py` 启动服务器
2. 确认访问地址是 `http://localhost:8084/index2.html`
3. 不要直接双击打开 HTML 文件

---

### 错误 2：CORS 错误

**错误信息：**
```
Access to fetch at 'file:///...' from origin 'null' has been blocked by CORS policy
```

**原因：** 直接打开了 HTML 文件，没有使用 HTTP 服务器

**解决方法：**
```bash
# 必须使用 HTTP 服务器
python start.py

# 然后在浏览器访问
http://localhost:8084/index2.html
```

---

### 错误 3：找不到 3D 容器

**错误信息：**
```
找不到 3D 容器
```

**原因：** 页面还没加载完就执行了脚本

**解决方法：** 刷新页面，等待完全加载

---

### 错误 4：Three.js 加载失败

**错误信息：**
```
Failed to load module script: Expected a JavaScript module script...
```

**原因：** 网络问题，无法加载 Three.js 库

**解决方法：** 检查网络连接，或者等待几秒后刷新

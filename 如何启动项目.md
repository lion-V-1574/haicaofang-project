# 海草房3D模型查看器 - 启动说明

## ⚠️ 重要提示

**不要直接双击打开 index2.html 文件！**

FBX模型文件需要通过HTTP服务器加载，直接打开HTML文件会导致加载失败。

## 正确的启动方式

### 方法1：使用Python服务器（推荐）

1. 确保已安装Python 3
2. 在项目根目录打开终端/命令提示符
3. 运行命令：
   ```bash
   python start.py
   ```
4. 浏览器会自动打开，或手动访问：http://127.0.0.1:8084

### 方法2：使用Python内置服务器

在项目根目录运行：
```bash
python -m http.server 8000
```
然后访问：http://localhost:8000/index2.html

### 方法3：使用Node.js服务器

如果安装了Node.js，可以使用：
```bash
npx http-server -p 8000
```
然后访问：http://localhost:8000/index2.html

## 常见问题

### Q: 为什么直接打开HTML文件不行？
A: 浏览器的安全策略（CORS）不允许从 file:// 协议加载外部资源（如FBX文件）。

### Q: 模型显示为黑色怎么办？
A: 这是正常的，模型加载后会自动应用颜色。如果一直是黑色，请检查浏览器控制台的错误信息。

### Q: 如何查看错误信息？
A: 按F12打开浏览器开发者工具，查看Console（控制台）标签页。

## 模型文件位置

确保以下文件存在：
- `images/model.fbx` - 海草房3D模型文件

## 技术支持

如果遇到问题，请检查：
1. 是否通过HTTP服务器访问（URL应该是 http:// 开头）
2. 浏览器控制台是否有错误信息
3. model.fbx 文件是否存在于 images 文件夹中
